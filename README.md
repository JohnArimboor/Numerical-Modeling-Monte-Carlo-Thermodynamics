# Numerical-Modeling-Monte-Carlo-Thermodynamics

Project Overview:- 
This project aims to use pyhton to build a numericaal simulation that calculates the macroscopic thermodynamic properties of real gases from their microscopic molecular interactions. By applying Monte Carlo integration, the simulation solves non-analytical intermolecular potentials (Lennard-Jones is the model used) to predict how different gases behave under high pressure.


Physics and equations used:-
